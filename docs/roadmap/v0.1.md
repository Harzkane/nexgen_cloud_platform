# Sprint Roadmap: v0.1 — Foundation

The goal of Version 0.1 is to establish a secure, production-grade OS foundation on an Ubuntu server.

---

## 🎯 Target Goal

> Turn a fresh, vanilla Ubuntu server VM into a secure, docker-enabled platform ready to host application stacks, completely automated with a single command.

---

## 📦 Scope & Deliverables

### Command
```bash
ncp install
```

### Execution Steps
1. **System Detection & Preparation:**
   - Detect OS version (must be Ubuntu 24.04 LTS or compatible).
   - Sync and upgrade standard system packages.
   - Install base system utilities (`git`, `curl`, `build-essential`, `htop`, `tree`, `zip`, `unzip`, `vim`, `nano`).
2. **Security Hardening:**
   - Enforce SSH key-based authentication.
   - Disable root password login.
   - Set up and configure the **Uncomplicated Firewall (UFW)** with defaults (deny incoming, allow outgoing) and open ports `22` (SSH), `80` (HTTP), and `443` (HTTPS).
   - Configure **Fail2Ban** to block brute-force attacks on SSH.
3. **Platform Initialization:**
   - Install Docker Engine & Docker Compose (latest stable release).
   - Setup standard directory structures under `/opt/nexgen/`.
4. **Verification & Reporting:**
   - Run verification checks on services (Docker running, firewall rules active, fail2ban active).
   - Generate a JSON/Markdown installation report showing status, package versions, and elapsed time.

---

## 🛠️ Verification Criteria

| Verification Check | Target Status | Expected Output |
|--------------------|---------------|-----------------|
| System Utilities   | ✅ Present    | git, curl, zip, etc. respond with versions |
| SSH Security       | ✅ Active     | Password authentication disabled in `sshd_config` |
| Firewall (UFW)     | ✅ Active     | Ports 22, 80, 443 open; default deny incoming |
| Fail2Ban           | ✅ Active     | service status: active (running) |
| Docker Platform    | ✅ Active     | docker and docker-compose respond with versions |
| Directory Layout   | ✅ Created    | `/opt/nexgen/` directory tree exists |
